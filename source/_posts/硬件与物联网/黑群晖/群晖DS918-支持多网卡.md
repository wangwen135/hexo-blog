黑群晖DS918 默认只支持2个网卡，为了验证一些问题，需要让其支持更多的网卡



### 修改说明

1、 SSH 登录群晖，切换的root用户
```
sudo -i
```
输入密码

2、修改synoinfo.conf配置文件

```
# vi /etc.defaults/synoinfo.conf 

# 找到
# maxlanport="2"  # 这个2就表示支持2个网卡，我们这里改成4

# 保存退出并重启
reboot
```
